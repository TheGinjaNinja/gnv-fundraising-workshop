# Setup Guide

## Option 1: Claude Code (recommended)

1. Create a Claude account at [claude.ai](https://claude.ai) (Max plan or API credits).
2. Open your terminal (Mac: Terminal. Windows: PowerShell).
3. Install Claude Code: `npm install -g @anthropic-ai/claude-code`
4. Clone this repo: `git clone https://github.com/TheGinjaNinja/gnv-fundraising-workshop.git`
5. Open it: `cd gnv-fundraising-workshop && claude`
6. When prompted, allow file read/write access.

For detailed setup with screenshots, see the [GNV Team Claude Code Setup Guide](https://github.com/TheGinjaNinja/gnv-team-claude-setup).

## Option 2: Claude Web (no installation)

1. Go to [claude.ai](https://claude.ai) and start a project.
2. Upload the files from `/knowledge/` and copy the contents of CLAUDE.md into the project instructions.

Note: outputs will not save to files automatically. Copy them manually.

## Option 3: Other AI tools

Works with Cursor, GitHub Copilot, Windsurf, or ChatGPT.

1. Copy the CLAUDE.md content into your system prompt or project instructions.
2. Upload the knowledge files as context.

## Option 4: Templates + AI Prompts (no installation)

1. Download the files from `/low-tech/`.
2. Open `AI-PROMPTS.md` and copy the prompt for your exercise.
3. Paste it into ChatGPT, Claude, or any AI tool and start the conversation.
4. Use the PowerPoint/Excel templates alongside the AI to capture your answers.
